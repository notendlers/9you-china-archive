# 9you-china-archive

本仓库用于归档和发布多个独立的 HTML 页面。  
内容以静态网页形式存放，不针对任何特定域名或具体网站。

## 项目简介

`9you-china-archive` 是一个以 HTML 页面为主的归档仓库。  
仓库中的页面彼此独立，可单独浏览或引用，主要用于保存和共享制作完成的静态网页资源。

## 目录说明

仓库目录结构如下：

```
9you-china-archive/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   └── example2.html
└── assets/             # 页面所需的资源文件（如图片、样式、脚本）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有归档的 HTML 页面均存放于此，每个文件为一个独立页面。
- `assets/`：页面引用的公共或私有资源，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖仓库外的资源。
- 页面内容保持原始状态，除非必要，不进行修改或更新。
- 归档范围覆盖多个来源的静态页面，不限于特定主题或领域。

## 使用方式

您可以通过以下方式查看或使用仓库中的页面：

1. 直接打开 `pages/` 目录下的 `.html` 文件（在本地浏览器中）。
2. 通过 GitHub Pages 或其他静态托管服务部署后在线访问。
3. 克隆或下载仓库到本地，按需浏览或引用。
